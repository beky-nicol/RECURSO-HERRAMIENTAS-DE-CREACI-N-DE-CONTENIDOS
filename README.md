**Recurso educativo digital: Componentes del Modelo Margarita - Animales Vertebrados**

Este proyecto presenta un crucigrama interactivo para conocer sobre los componentes del Modelo Margarita y un mapa multimedia diseñados para fortalecer el aprendizaje sobre los animales vertebrados El recurso promueve la exploración, el análisis y la comprensión activa mediante actividades visuales e interactivas, fomentando el pensamiento crítico y la integración de saberes en el proceso educativo.
